# 📊 Customer Demographics & RFM Analysis Dashboard | Power BI

## 📌 Project Overview

This project is my second Power BI dashboard, built using the AdventureWorks dataset.

The dashboard focuses on customer demographics, customer segmentation using RFM analysis, and detailed customer information. It demonstrates data modeling, DAX calculations, interactive visualizations, and dashboard design to provide meaningful business insights.
